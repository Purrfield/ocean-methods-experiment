\section{实验目的与安排}

\subsection{实验目的}

\subsubsection{海洋调查实践的意义}

海洋科学是一门实践性很强的学科，从科学角度来看，许多重大发现都是从观测调查中得来的。从现实情况来看，海洋资源与环境、海洋权益、国土安全、减灾防灾等关系到国计民生的大事，又无不与海洋监测密切相连。特别是随着海洋科学的发展，人们越来越意识到以空基、船基、陆基、海床基、自持自航设备等三维立体观测系统进行多学科、多参数高时空分辨率、长时间序列的同步观测为基础，高新技术手段的应用、可靠数据的获取是研究成果融入国际主流、建立世界一流独树一帜的理论学说的前提。中国目前正处于经济快速增长期，海洋事业蒸蒸日上，需要大量全面发展的海洋科技人才，因此培养学生的海上实践能力，意义重大。

\subsubsection{海上调查实践的目的}

海上实践的目的主要是要求学生了解海洋科学研究数据获取、分析的全过程，掌握部分观测仪器的使用要领，通过对海洋现象的感性认识，加深对海洋科学理论的理解，锻炼意志，培养学生认识问题、分析解决问题的能力，为其今后更好地从事海洋研究、海洋管理、环境保护等工作打好基础。近海海洋调查有其自身的特点，比如很少有专用的调查船，往往在研究海域当地租用渔船或其他船只，因此存在着仪器需要临时安装固定，电源不匹配，不能有良好的工作、休息条件等。需要调查的组织者、参与者有很好的动手、动脑以及与人协调的能力，通过本调查实践希望让学生能够对此有所体验。

\section{实验内容与数据}

\subsection{实验一：AUV自主操作实验}

\subsubsection{实验目的}
掌握无缆水下航行器（AUV）的工作方式、应用领域和设置方法，利用 AUV 进行自主操作实验，获取布放海域相关的海洋水文环境参数。

\subsubsection{实验内容}
在指定站位下放 AUV，自主操控 AUV 对目标海域进行自主航行观测，记录 AUV 下放经纬度、下放时间、航行轨迹等重要参数，获取航行海域相关水文环境参数。在不同深度层下，对 AUV 进行巡航测试，实现不同精度网格情况下的巡航。

\subsubsection{实验设备}
HN-660 型便携式 AUV 及配套设备。

\subsubsection{实验步骤}
\begin{enumerate}[label=（\arabic*）]
  \item 仪器自检与布放：布放前对 AUV 系统内部电气各个模块进行上电确认及通信测试，确保系统可以正常入水工作。自检尽量在室外 GPS 和 4G 信号良好的环境下试验，包括手机状态检查、平板状态检查和航行器状态检查等。检查结束后，利用绞车、吊机等设备将 AUV 吊放至海面，吊放时注意防止磕碰。
  \item 水面操作控制：自主航行开始前，进行水面遥控操作，验证系统推进器、舵机、姿态是否正常工作，工作状态是否稳定。
  \item 自主操控 AUV 完成航行与数据采集：根据调查任务及要求完成 AUV 的任务规划设置，监控 AUV 完成任务。AUV 任务主要包含直航任务和导航任务，学生可根据调查要求进行选择。
  \item 数据分析与 AUV 回收：将航行任务中记录的相关数据从手机导航端下载到平板操控端，将数据中的 GPS 位置信息显示在地图中。全部任务结束后，将 AUV 遥控至船边并捞起，进行清理冲洗、断电、存储等工作后，将 AUV 妥善收好。
\end{enumerate}

\subsection{实验二：湖泊表面光谱测量}

\subsubsection{实验目的}
掌握湖泊表面光谱测量方法，熟悉高光谱辐射计测量操作和数据处理过程，理解湖泊表面光谱反映的湖泊水体特征，思考湖泊表面光谱测量可能受哪些因素影响。

\subsubsection{实验内容}
在岸边利用光谱仪分别测量太阳向下辐照度、海水光谱辐亮度、天空光谱辐亮度，利用测量数据计算得到海洋遥感反射率（比）光谱。

\subsubsection{实验设备}
光谱仪、数据采集终端、GPS、半圆仪、长直尺、测量卷尺、记录本等。

\subsubsection{实验步骤}
\paragraph{准备工作}
\begin{enumerate}[label=（\arabic*）]
  \item 光谱仪电池、数据采集终端充电：光谱仪电池电量不足时红灯闪亮，充满电后绿灯亮；如果黄灯闪亮则说明过热，需要等待一段时间。
  \item 将光谱仪、电缆、数据采集终端连接后，依次打开光谱仪电源及数据采集终端电源，并启动相应数据采集软件。
  \item 在软件中选择设置定标文件、数据采集配置、存储路径、测量站位名称等。准备工作连接设置详细操作可参考 RAMSES 辐射计手持机用户手册。
\end{enumerate}

\paragraph{光谱测量}
\begin{enumerate}[label=（\arabic*）,resume]
  \item 辐照度传感器垂直向上，测量获得向下太阳辐照度 \(E_d\)。
  \item 辐亮度镜头对准天空、水面，测量获得天空光谱辐亮度（\(L_{sky}\)）、水面光谱辐亮度（\(L_u\)）；常用测量几何如下图所示，一般选择 \(\theta_v=40^\circ\)，\(\phi_v=135^\circ\) 或 \(90^\circ\)，方法可参考文献唐军武等（2004）。
\end{enumerate}

\subsection{实验三：水质观测}

\subsubsection{实验目的}
掌握湖水温度、盐度、溶解氧等参数的测量方法和仪器的使用。

\subsubsection{实验内容}
利用多参数水质仪测量海水温度、盐度、溶解氧等数据。

\subsubsection{实验设备}
OT-Switch 多参数水质仪。

\subsubsection{实验步骤}
水质观测：利用多参数水质仪对湖泊的温度、盐度、溶解氧（DO）、叶绿素、pH 和浊度等参数进行剖面观测。

\begin{enumerate}[label=（\arabic*）]
  \item 首先对多参数水质仪进行设置，因进行剖面观测，因此采样频率宜采用 5Hz 或者 25Hz，具体操作指南见附录 3。
  \item 仪器设置完毕后，确保仪器正常工作，之后借助绞车等工具将多参数水质仪放至船边，保持匀速缓慢下降，从而获取站位相关参数的剖面数据。
  \item 下放过程中注意避免触底，避免仪器磕碰船舷。根据剖面数据绘制剖面曲线图、大面站所测水域水质参数空间分布图等，并尝试阐述曲线产生机理。
\end{enumerate}

\subsection{实验五：海洋牧场低氧灾害调查监测虚拟仿真实验}

\subsubsection{实验目的}
\begin{enumerate}[label=\arabic*)]
  \item 了解海洋牧场概念及意义，了解海洋牧场可能发生的低氧或赤潮等海洋灾害及危害。
  \item 认识并掌握海洋牧场以及海底海洋环境实时监测的相关知识，学习掌握海底有缆观测系统布放、海洋牧场数据采集与维护等现场调查操作步骤，巩固海洋调查知识。
  \item 对常见的海洋调查仪器进行学习，认识仪器外观并掌握仪器功能，加深理解海底在线观测的优点和难点。
\end{enumerate}

\subsubsection{实验内容}
实验包括知识学习、设备展示、设备布放三个模块，其中：
\begin{enumerate}[label=\arabic*)]
  \item 知识学习模块对实验背景知识进行学习，对实验内容进行初步了解。
  \item 设备展示模块对实验中的调查仪器、观测系统进行展示，对设备的组成、功能进行简单介绍。
  \item 设备布放模块通过以真实的海洋牧场为原型进行实际的海底在线观测系统布放流程仿真模拟学习，利用掌握的知识点进行交互现场观测设计，实现对知识掌握情况自我检测。
\end{enumerate}

\subsection{实验六：连续水位与海流观测}

\subsubsection{实验目的}
掌握基于海床基观测系统的长期连续水位、海流观测方法和仪器的使用。

\subsubsection{实验内容}
在指定站位布放海床基观测系统，系统搭载潮位计与多普勒声学流速剖面仪，记录布放经纬度、布放时间及仪器设置参数，获取连续时间的水位与海流观测数据，并基于实测数据绘制水流与海流变化折线图。

\subsubsection{实验设备}
海床基观测系统、UMI-OT 潮位计、诺泰克“阔龙”声学流速剖面仪。

\subsubsection{实验步骤}
\begin{enumerate}[label=（\arabic*）]
  \item 仪器设置：按连续观测要求设置潮位计和声学流速剖面仪，潮位计操作要求与实验三中的多参数水质仪类似，具体操作指南见附录三：《国产多参数操作指南》，声学流速剖面仪操作指南见附录四：《诺泰克“阔龙”操作指南》。
  \item 系统组装搭建：利用尼龙件、常平架等将潮位计、声学流速剖面仪固定于海床基上，固定仪器时注意不要过松或过紧，尤其注意常平架的安装，组装后的海床基观测系统如图所示。
  \item 在指定站位区域，利用绞车、吊机等设备布放海床基观测系统，布放区域尽量避开淤泥底质，海底地形平坦，布放过程注意安全，在甲板作业时佩戴安全帽，海况超过 3 级时，不宜开展海床基观测系统布放作业。

\subsection{实验七：定位与气象观测}

\subsubsection{实验目的}
掌握常规数据的定义及观测记录方式、方法。

\subsubsection{实验内容}
记录实验当天各站点的水深、能见度、风速、风向、大气温度、海况、波型、气压、位置数据，风速风向在相对坐标系和绝对坐标系之间相互转换。

\subsubsection{实验设备}
船用气象仪、水深探测系统。

\subsubsection{实验步骤}
\begin{enumerate}[label=（\arabic*）]
  \item 海上能见度观测：参考水平能见度观测表对海上能见度进行观测及记录，观测有效水平能见度和最小水平能见度。有效水平能见度是指测站四周 1/2 以上的范围所能看到的最大水平距离。最小水平能见度是指测站四周在能见度不一致时所能看到的最小水平距离。能见度以 km 为单位，取小数一位，准确度为 20\%。
  \item 气象观测：手持风速风向仪。风速以 m/s 为单位，读数分辨至 0.1m/s，风速 \(\leq 5\)m/s，准确度为 \(\pm 0.5\)m/s，\(>5\)m/s 时，准确度为 \(\pm 10\%\)；风向以度为单位，读数分辨率至 \(1^\circ\)，准确度 \(\pm 5^\circ\)，正北为 \(0^\circ\)，顺时针计量，观测应在迎风一侧。气压指海面平均气压，单位为百帕（hPa），分辨率为 0.1hPa，准确度为 \(\pm 1\)hPa。
  \item 水深海况观测：船舶驾驶室装有水深探测系统，可读取水深数据；目测海浪时，应站在船只迎风面，以离船身 30m（或船长之半）以外的海面作为观测区域（同时还应环视广阔海面）来估计波浪尺寸和判断海浪外貌特征。以目力观测海面征象，根据海面上波峰的形状、峰顶的破碎程度和浪花出现的多少，按海况等级判断海况所属等级，并填入记录表中。
\end{enumerate}

\subsection{实验八：水文、水质观测}

\subsubsection{实验目的}
掌握海水透明度、水色的定义及观测方法；掌握海水温度、盐度、溶解氧等参数的测量方法和仪器的使用。

\subsubsection{实验内容}
记录实验当天各站点的海水透明度、水色，并利用多参数水质仪测量海水温度、盐度、溶解氧等数据。

\subsubsection{实验设备}
透明度盘、长绳、水色比色计、采水器、OT-Switch 多参数水质仪。

\subsubsection{实验步骤}
\begin{enumerate}[label=（\arabic*）]
  \item 海水透明度观测：观测应在甲板的背阳处，将透明度盘垂直放入水中，沉到刚好看不见的深度，然后再慢慢地提升到透明度盘隐约可见时，读取绳索在水面的标记数值（有波浪时应分别读取绳索在波峰和波谷处的标记数值，重复三次，求其平均值）；读到一位小数，重复二到三次，取其平均值，即为观测的透明度值，计入观测记录表中。若倾角超过 \(15^\circ\)，则应进行深度订正。当绳索倾角过大时，盘下的重锤应适当加重。
  \item 水色观测：透明度观测结束后，将透明度盘提升到透明度值一半的水层，根据透明度盘上所呈现的海水颜色，在水色计中找出与之最相似的色级号码，即为该次观测的水色，并记入观测记录表中，观测时观测者的视线需保持与水色计玻璃管垂直。
  \item 水质观测：利用多参数水质仪对站位的温度、盐度、溶解氧（DO）、叶绿素、pH 和浊度等参数进行剖面观测。首先对多参数水质仪进行设置，因进行剖面观测，因此采样频率宜采用 5Hz 或者 25Hz，具体操作指南见附录 3。仪器设置完毕后，确保仪器正常工作，之后借助绞车等工具将多参数水质仪放至船边，保持匀速缓慢下降，从而获取站位相关参数的剖面数据。下放过程中注意避免触底，避免仪器磕碰船舷。根据剖面数据绘制剖面曲线图、大面站所测海域水质参数空间分布图等，并尝试阐述曲线产生机理。
\end{enumerate}

\subsection{实验九：海洋表面光谱测量}

\subsubsection{实验目的}
掌握海洋表面光谱测量方法，熟悉高光谱辐射计测量操作和数据处理过程，理解海洋表面光谱反映的海洋水体特征，思考海洋表面光谱测量可能受哪些因素影响。

\subsubsection{实验内容}
在指定站位选择岸边、湖边、或者调查船舷处，利用光谱仪分别测量太阳向下辐照度、海水光谱辐亮度、天空光谱辐亮度，利用测量数据计算得到海洋遥感反射率（比）光谱。

\subsubsection{实验设备}
光谱仪、数据采集终端、GPS、半圆仪、长直尺、测量卷尺、记录本等。

\subsubsection{实验步骤}
\paragraph{准备工作}
\begin{enumerate}[label=（\arabic*）]
  \item 光谱仪电池、数据采集终端充电：光谱仪电池电量不足时红灯闪亮，充满电后绿灯亮；如果黄灯闪亮则说明过热，需要等待一段时间。
  \item 将光谱仪、电缆、数据采集终端连接后，依次打开光谱仪电源及数据采集终端电源，并启动相应数据采集软件。
  \item 在软件中选择设置定标文件、数据采集配置、存储路径、测量站位名称等。准备工作连接设置详细操作可参考 RAMSES 辐射计手持机用户手册。
\end{enumerate}

\paragraph{光谱测量}
\begin{enumerate}[label=（\arabic*）,resume]
  \item 辐照度传感器垂直向上，测量获得向下太阳辐照度 \(E_d\)。
  \item 辐亮度镜头对准天空、水面，测量获得天空光谱辐亮度（\(L_{sky}\)）、水面光谱辐亮度（\(L_u\)）；常用测量几何如下图所示，一般选择 \(\theta_v=40^\circ\)，\(\phi_v=135^\circ\) 或 \(90^\circ\)，方法可参考文献唐军武等（2004）。
\end{enumerate}
